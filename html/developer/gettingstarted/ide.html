Get an IDE

As GlobalTester is based on Eclipse RCP we also use Eclipse as our IDE. Go get a recent package from http://www.eclipse.org/downloads/. Just unzip it to a location of your choice and fire up the executable inside.

Required Plugins

To get the code and handle it you will need at least the following plugins:

  • PDE (this may be already included, depending on the package you chose to download)
  • Buckminster
  • Subversion integration (choose whichever you want, we use Subversive)

Optional plugins

  • WebPageEditor (for editing documentation)
  • JUnit (for unit testing)
  • SWTBot (for testing UI)
  • WindowBuilder (for easier design of UI)
