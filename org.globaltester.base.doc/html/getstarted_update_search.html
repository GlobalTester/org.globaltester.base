Manually Updating GlobalTester

In order to search for updates of GlobalTester Plug-ins, you call the Software Manager by selecting Help in the main menu of Eclipse and then click Installed Software. The window Eclipse SDK Installation Details with the Installed Software tab will pop up:

Select the GlobalTester features you want to update and press Update …. The Eclipse Update Manager will check for updates at the referenced update site. You will be asked for your username and password, which you have received from us. Please enter username and password and continue.

If there are updates, the Update Manager will ask you whether to install these updates.
