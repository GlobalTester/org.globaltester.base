Checking that GlobalTester was successfully installed

After restarting Eclipse you can confirm that your selected GlobalTester products are successfully installed.

Firstly you will see the buttons Script execution and TestCampaign creation in the main menu:

 

Additionally you can view the list of plug-ins which are installed. Select Help, Installation Details

 

The Installed Software tab should be selected. The newly installed Plugin group should be displayed. The exact name of it depends on your installation. In the screenshot below you see the "GlobalTester Prove EPP".

 

Additionally you can check the Features tab. Just scroll down in the list of features which are installed and you will see your installed feature.
