Support GlobalTester

The GlobalTester is released as Open Source software on GitHub. General information about the product can be found at www.globaltester.org. If you need support please create an issue on https://github.com/GlobalTester.

Demo test cases

If you would like to receive demonstration test scripts for ePassports (ICAO) please contact us at globaltester@hjp-consulting.com and we will send you these via mail.

Feedback

The team of GlobalTester developers is always interested in your feedback. If you want to send some feedback, please feel free to contact the team via eMail globaltester@hjp-consulting.com.

Newsletter

To subscribe the GlobalTester newsletter please to to the following site at sourceforge: https://lists.sourceforge.net/lists/listinfo/globaltester-news.
