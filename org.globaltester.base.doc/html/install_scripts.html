Install scripts


To install test scripts you can choose between the following two options:

  • import from separate download file
  • import from already installed product

 

Import from separate download file

To install test scripts you will need to download the appropriate ZIP-file first and store it in a location you can remember.

When you started Eclipse with an empty workspace and switched to GlobalTester perspective either select File, Import or right-click in the GlobalTester Navigator and select Import. The following dialog will appear:

Select General, Existing Projects into Workspace and click the Next button.

Choose Select archive file: and browse to the archive you downloaded before.

Then select the projects to import and click Finish

When the projects are imported successfully you will see a new project in GlobalTester Navigator.

 

 

import from already installed product

When you started Eclipse with an empty workspace but already installed a plug-in product and switched to GlobalTester perspective either select File, Import or right-click in the GlobalTester Navigator and select Import. The following dialog will appear:
