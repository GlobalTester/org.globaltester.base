Installing GlobalTester test scripts

After successful installation of Eclipse and the GlobalTester Plug-ins all you need to run tests are test scripts that define test suites and appropriate test cases.

To develop your own test scripts see Develop GlobalTester test scripts.

For BSI and ICAO conform testing of electronic passports HJP Consulting offers test scripts within their GT Prove ePP product. Likewise HJP Consulting offers a product for testing inspection systems, called GT Prove IS.
