Installing GlobalTester Plug-ins

The preferred way of installing GlobalTester Plug-ins is to use the online update site. This is very convenient as it automatically does the dependency resolution and enables the possibility for automatic updates. Nevertheless you can install the plug-ins manually, see the section called “Installing GlobalTester Plug-ins offline” to find detailed instructions.

GlobalTester Plug-ins are installed by using the standard Eclipse Install Software mechanism.

From the Eclipse main menu select Help, Install New Software...

The dialog Available Software will appear. Just click the Add... button to add a new Update Site.

You will then be prompted to select the location which should be searched for new software. The URL you need to enter here depends on your subscription. You can use one of the following:

  • https://www.globaltester.org/globaltester/update

  • https://www.hjp-consulting.com/globaltester/gtproveepp/update

  • https://www.hjp-consulting.com/globaltester/gtproveepa/update

  • https://www.hjp-consulting.com/globaltester/gtproveis/update

For the later three you will need a valid username and password which you receive together with your subscription to the according products. After making these entries click OK.

Note

This is a secure site with a URL prefix HTTPS. Due to a bug concerning Eclipse together with Java 6 versions 1.6.0_7 to 1.6.0_13, update sites using HTTPS may not work in some situations. In this case you should upgrade to JDK version 1.6.0_14 or use the installation via local update site as described in the section called “Installing GlobalTester Plug-ins offline”. You may also need to install "Unlimited Strength Jurisdiction Policy Files". You can download them directly from Oracle. Please make sure to download a version that fits with your installed Java Runtime Environment. Just copy the files into \Java\jreX\lib\security folder and overwrite the existing files.

Now you see your new update site available and you can select which features to load from the side. In the screenshot below you see an Eclipse installation an example update site. The Features you can select depend on the chosen update site. Some of them may not be accessible for you depending on your subscription. Select the features you want to use. Now press the Next button.

The following dialog shows you which packages need to be installed for installing. Press Next to accept this.

You will then be prompted to agree to the license conditions for GlobalTester. The installation is started by clicking on Finish.
