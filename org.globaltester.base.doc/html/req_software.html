Software

The GlobalTester framework generally requires the following software environment:

  • Windows 7 or newer

  • Java Development Kit 1.7 or higher

  • JCE Unlimited Strength Policy files (make sure to install the fitting version for your installed JDK)

  • Eclipse 3.6 or higher

  • GlobalTester Plug-ins

  • a viewer for PDF-files to show reports

It is recommended that you install the latest version of Eclipse – currently Version 3.4 and the latest versions of the GlobalTester Plug-ins.

At least the GT TestManager Plug-in is required to use GlobalTester. This depends on the GT Logging and GT OnlineHelp Plug-ins. When you use the GlobalTester Update Site for installation, as described in the section called “Installing GlobalTester Plug-ins”, you don't have to care about this, Eclipse will handle that for you.

If you want to test e-passports you will need the GT Prove ePP Plug-in. This depends on GT TestManager and extends it with the needed functionality to perform layer 6 and layer 7 tests of ICAO conform e-passports.

To test inspection systems for e-passports you will need the GT Prove IS Plug-in. This depends on GT TestManager and extends it with functionalities to access an e-passport Simulator. Using the simulator both valid and invalid communications can be tested with the inspection system and GlobalTester will generate detailed logfiles and reports.

HJP Consulting offers packages for each of the above use cases that contain all the needed plug-ins.
