Installing GlobalTester Plug-ins manually

If you want to install GT Plug-ins to an existing workspace, e.g. if you are already using a GT deploy bundle and acquired an additional plug-in, you can install the plug-ins manually.

For manual installation of Global Tester framework you can download a ZIP-file containing a copy of the update site from http://www.globaltester.org or (depending on your subscription) from our download site (remember to Login or the downloads are hidden). This ZIP-file includes all files that are present on the update site together with the needed meta files to create a local update site.

To install plug-ins from the ZIP-file simply unpack the whole archive into a location in your local filesystem and select this location as update site during the installation process as described in the section called “Installing GlobalTester Plug-ins”.

When updates to your purchased products become available new update site images will be provided which you can unpack in the same location in your filesystem and overwrite old files.
